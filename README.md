# PeerSend

PeerSend is a peer-to-peer file transfer project for desktop and mobile.

It is designed to make file transfer feel simple and direct between PCs and phones with:

- LAN mode for nearby devices on the same network
- Tunnel mode for remote device-to-device connections
- Desktop engine + web UI architecture
- Android and iOS mobile apps
